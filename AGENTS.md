# rlm-pi — Project Instructions

- This project uses **bun.js** as the runtime and package manager.
- TypeScript source lives at the root and in `src/`.
- Run `bun run index.ts` to execute the Pi agent harness.
- Use `bun install` to add dependencies — do NOT use npm/pnpm/yarn.
- Keep responses concise and follow existing code style.
- Do not commit secrets, API keys, or session files.
